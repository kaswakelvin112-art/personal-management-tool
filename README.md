# Weekly Schedule Manager with Reminders

A simple web-based application for managing and viewing a weekly schedule with built-in reminders and real-time activity tracking.

## Features

- **Weekly Schedule View**: Displays a complete weekly schedule with activities organized by day
- **Real-time Clock**: Shows the current time and highlights the current activity
- **Activity Categories**: Activities are categorized as Study, Class, or Break for easy identification
- **Reminders**: Built-in reminder system for scheduled activities
- **Responsive Design**: Clean, modern interface that works on desktop and mobile devices
- **Live Activity Status**: Displays the current ongoing activity with countdown timer

## Technologies Used

- HTML5
- CSS3 (with gradients and modern styling)
- Vanilla JavaScript (no frameworks required)

## How to Use

1. Clone or download the repository
2. Open `index.html` in any modern web browser
3. The schedule will load automatically with predefined activities
4. The app will show the current time and highlight the active activity

## Schedule Overview

The application includes a sample schedule for a student with activities such as:

- Morning routines and breakfast
- Academic classes (AI Mathematics, Software Development Principles)
- Study sessions (Machine Learning, lecture reviews)
- Breaks and meals
- Evening relaxation

## Customization

To customize the schedule, edit the `scheduleData` array in `index.js` with your own activities, times, and categories.

## Browser Support

Works in all modern browsers that support ES6 JavaScript features.
